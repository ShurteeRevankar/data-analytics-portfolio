# Superstore Sales Analysis (Tableau)

## Objective
Analyse sales and profit performance  of a retail superstore using Tableau and identify key business insights.

## Tools
- Tableau Public
- Excel (for data cleaning)

## Dataset
Sample Superstore dataset

## Dashboards
1.	Executive Overview
2.	Geographical Performance
3.	Customer Analysis
4.	Sales & Profit Analysis
5.	Superstore Sales Dashboard

## Key Insights
- Technology category has highest profit.
- Some sub-categories show losses despite high sales.
- West region performs best overall.
