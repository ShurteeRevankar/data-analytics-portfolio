# Superstore Analysis

## Objective
Analyse sales and profit performance  of a retail superstore using Tableau and identify key business insights.

## Tools
- Tableau Public
- Excel (for data cleaning)

## Dataset
Sample Superstore dataset

## Dashboards
1.	Superstore Sales Dashboard

## Key Insights
- Technology category has highest sales.
- Phones have highest sales compared to other sub-categories
- Some sub-categories show losses despite high sales.
- Profit Margin is 12.47%.
- Consumer segments shows highest sales.
