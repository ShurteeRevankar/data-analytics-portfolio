# Zomato Analysis

## Objective: 
Analyze restaurant data from Zomato to understand cuisines, ratings, and geographical distribution, and derive key business insights.

## Tools:
- Tableau
- Power BI
- Excel

## Zomato Restaurant Dataset:
- Country code dataset
- zomato dataset

### Dashboard: 
- Executive Overview
- Cuisine Analysis
- Ratings Distribution
- Country-wise Restaurant Analysis
- Average Ratings by Cuisine

## Key Insights:
- Total restaurants analyzed: 9551
- India has the highest number of restaurants
- North Indian cuisine is the most popular
- Most ratings fall between 3 to 4 range
- Online delivery is more common than table booking
