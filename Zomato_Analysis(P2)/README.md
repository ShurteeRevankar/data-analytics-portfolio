# Zomato Analysis

## Objective
Analyse restaurant distribution, customer preferences, pricing segments, and growth trends using data visualization.

## Tools

- Power BI
- Excel
- SQL Workbench

## Dataset
Zomato dataset

## Dashboards
1.	Zomato Analysis Dashboard - Tableau
2.	Zomato Analysis Dashboard - Power BI
3.	Zomato Restuarant Analysis - Excel

## Analysis Presentation - PPT

## Key Insights
- Majority of restaurants fall in the affordable and mid-range category, indicating Zomato mainly targets mass consumers.
- Very few premium restaurants are listed, showing limited luxury dining focus.
- Most restaurants have ratings between 3.5 and 4.0, indicating average-to-good quality.
- North Indian cuisine dominates, showing strong regional food preference.
- Online delivery adoption is higher than table booking, highlighting customer preference for convenience.
