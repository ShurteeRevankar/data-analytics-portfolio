I have built three dashboards to visualize performance. 

1. The Excel Dashboard covers overall hospitality performance – it includes monthly revenue trends, room class analysis, revenue by city, booking status breakdown, and top performing hotels.
2. The Tableau Dashboard focuses on weekly trends, showing weekly occupancy patterns and a weekday versus weekend revenue comparison.
3. The Power BI Dashboard is a deep-dive into the Elite room class, covering metrics like RevPAR, ADR, Occupancy Percentage, and Realization Rate, along with a property-wise breakdown table. 

- Together, these dashboards convert complex data into clear, actionable insights
